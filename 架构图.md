## 模块职责

| 模块 | 输入 | 输出 | 类型 |
|------|------|------|------|
| `prompt_builder` | 用户输入 + 任务上下文 | 完整 prompt 字符串 | 确定性代码 |
| `mock_model` | prompt 字符串 | `{"action", "args"}` dict | **模拟模型** (可替换为 LLM) |
| `parser` | 模型原始输出 | 规范化 `{"action", "args"}` | 确定性代码 |
| `guardrail` | 解析后的 action | `(通过, 原因)` | 确定性代码 (安全策略) |
| `tool_executor` | action + args | `{"success", "result", "error"}` | 确定性代码 |
| `trace_logger` | 事件 dict | 追加写入 trace.jsonl | 确定性代码 |
| `run_eval` | 评测用例集 | 准确率统计 | 确定性代码 |

## Human-in-the-Loop 流程

```
危险操作 (delete_all_tasks)
        │
        ▼
  guardrail 拦截
        │
        ▼
  输出警告 + 等待用户输入
        │
    ┌───┴───┐
    ▼       ▼
  确认(y)  拒绝(n)
    │       │
    ▼       ▼
  执行     返回错误
 操作     "用户拒绝"
```
